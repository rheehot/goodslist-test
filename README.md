# goodslist-test

상품리스트 화면

## 결과물 확인

https://keen-benz-9ba5b0.netlify.com/

## 사용한 라이브러리

- react
- styled-components
- mobx
- jest

## 애플리케이션 실행 방법

`yarn start` or `npm start`

`http://localhost:8888/`에서 확인 가능합니다.

## 테스트 실행 방법

`yarn test` or `npm test`

**테스트 기능**

- 리스트 필터 테스트
- api에서 내려오는 모집률이 실제로 값을 계산하였을때 비슷한지 테스트
